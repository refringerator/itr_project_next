#

Full-stack Web-app for personal collection management

### Cloud platforms

1. supabase
2. vercel
3. github

### Tech stack

1. nodejs
2. nextjs (vercel)
3. prisma
4. postgres (supabase)
5. s3 like storage (supabase)
6. realtime (supabase)
7. auth (supabase)
8. fulltext search (postgres ext.)
9. ant design
